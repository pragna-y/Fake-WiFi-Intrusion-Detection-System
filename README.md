# Fake-WiFi-Intrusion-Detection-System

A Python and Streamlit-based intrusion detection system that identifies potentially malicious wireless networks using rule-based security analysis and risk scoring.

## Features

- Duplicate SSID detection
- Open network identification
- Weak encryption (WEP) detection
- Risk score generation
- Safe, Suspicious, and Fake network classification
- Interactive Streamlit dashboard
- Network risk visualization

## Tech Stack

- Python
- Streamlit
- Pandas
- Matplotlib

## Concepts Demonstrated

- Intrusion Detection
- Security Analytics
- Risk Scoring
- Data Visualization
- Dashboard Development
